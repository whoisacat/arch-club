# Деньги (Money)

## Представляет денежное значение

| **Money**                           |
|-------------------------------------|
| amount, currency                    | 
| **Operations**                      |
| +, -, *, allocate, >, <, <=, >=, =  |
* В основе типового решения деньги лежит идея создания класса, поля которого будут
  содержать величину денежной суммы, а также валюту, в которой указана эта сумма.
* Величина денежной суммы может быть представлена значением целочисленного типа или 
  же действительным значением с фиксированным количеством десятичных знаков.
* Методы проверки на равенство и хэш-операции соответствующих объектов долж-
  ны быть переопределены так, чтобы их выполнение основывалось на величине денежной
  суммы и используемой валюте.
